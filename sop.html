Video Delivery Process

How Clients Submit Videos

Clients are instructed to create a Google Drive folder named First Name Last Name - Company Name, set it to Editor access, and share the link in Slack.


VA Review Checklist

When a client drops a Drive link in Slack, verify:

  1. Folder is named correctly.
  2. Sharing is set to Editor (not Viewer).
  3. Only one video per script is uploaded.
  4. Files are named EXACTLY as the script names.

If anything is wrong, reply in Slack immediately. If correct, move to the production queue.

Cloudflare Access SOP

How to Grant University Access to a New Client

When a new client onboards, they need access to layerengine.ai/university. This is protected by Cloudflare Zero Trust.

  1. Log into dash.cloudflare.com
  2. Navigate to Zero Trust -> Access -> Applications
  3. Click Edit on the "LayerEngine University" application.
  4. Scroll down to the Policies section and click Edit on the active policy.
  5. Scroll to the Include section (where it says "Emails").
  6. Type the client's email address into the field and press Enter/Add.
  7. Click Save policy.

The client can now access the university using their email address.
